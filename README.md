# AI-Driven Citizen Grievance & Sentiment Analysis System

## 🔹 Week 1: N-gram Analysis

N-grams are sequences of words used to identify common patterns in citizen complaints.

- **Unigrams** show the most frequent individual words such as *noise* and *parking*.
- **Bigrams** capture common two-word phrases like *illegal parking* and *loud music*.
- **Trigrams** provide more context with phrases such as *commercial overnight parking*.

This analysis helps identify the most common civic issues, allowing government departments to understand and prioritize problems more effectively.

## 🔹 Week 2: TF-IDF Feature Extraction

TF-IDF (Term Frequency - Inverse Document Frequency) is used to convert text data into numerical form.

- It gives higher importance to meaningful and relevant words
- Reduces the importance of very common words
- Helps machine learning models understand text data

Using TF-IDF, complaint descriptions are transformed into feature vectors, which are used to train models for automatic classification of complaints into different departments.
